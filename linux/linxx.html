day1

Start now：
目录结构
  •  /bin  二进制可执行文件 所有用户
  •  /sbin  二进制可执行文件 root用户
  •  /boot  系统启动核心文件
  • /dev  设备目录
  • /home  用户主目录
  • /root   root 用户主目录
  •  /etc   系统配置文件
  •  /etc/passwd   用户信息文件
  •  /etc/group   用户组信息文件
  •  /mnt   挂载目录
  •  /lib   系统共享动态链接库
  •  /var   存放经常发生变化的文件   如缓存,登陆文件，mysql数据库的文件等
  •  /proc 　/sys   虚拟目录，内存映射，可以访问内存中的系统信息
  •  /usr  存放应用程序和文件
目录操作
  • mkdir  创建目录
  • mkdir -p  递归创建目录
  • rmdir  删除空目录
  • rmdir -p  递归删除空目录
  • rm -r  删除目录+内容(有确认选择)
  • rm -rf  删除目录和内容，无提示
  • rm file  删除文件
  • mv dir1 dir2  dir1改名和移动为dir2
  • cp -r dir1 dir2  复制    如果是目录这复制dir1到dir2下    如果是文件则复制后更名为dir2
  • cd ~ 或 cd / 或 cd   切换到用户主目录
  • cd - 切换到上次目录
文件操作
  • touch  创建一个空文件
  • cat  查看文件内容
  • more less  多页显示内容
  • head|tail -n  从前或后显示n条
  • tail -f  动态显示文件内容
  • wc  统计文件内容行、句、字符数
  • 添加内容到文件   > 覆盖写 >> 追加写
用户和组操作
  • useradd  添加用户
    • -g  组id
    • -d  用户主目录
    • -u  用户id eg: useradd -g 502 -d /shabi -u 502 dashabi

  • usermod  修改用户
    • -l  修改用户名

  • userdel  删除用户(信息）
    • -r  同时删除用户目录

  • groupadd  添加组
  • groupmod  修改组
    • -n  新的组名 旧组名

  • groupdel 删除组(有用户的组无法删除)

  • passwd  修改当前用户的密码

  • passwd  用户名 修改指定用户的密码(root特权)
  • /etc/passwd 用户信息
  • /etc/group 组信息

vi和vim使用
  •  命令模式
  •  尾行模式
  • 插入模式

尾行==>>插入

  • a   光标往后移动一格
  • i   什么事都没发生
  • o   当前行后插入一行空白行
  • s   删除当前光标位置的字符

尾行模式：

  • :set nu  显示行号
  • :set nonu | :set nu!  不显示行号
  • :w  保存文档
  • :q  退出
  • :q!  强制退出不保存
  • :wq!  强制保存后退出
  • :x  保存并退出，没修改就直接退出
  • :n  跳转到n行
  • :s/xx/yy/  替换光标所在行的第一个xx为yy
  • :s/xx/yy/g  替换一行的全部xx为yy
  • :%s/xx/yy/g  替换全部xx为yy
  • :%s表示所有行,8s表示第8行,8,10s表示第8-10行
  • :syntax on/off  打开 | 关闭 语法高亮

命令模式

  • h、j、k、l  左、下、上、右
  • dd  删除当前行
  • ndd  删除当前行往下n行
  • x 或 delete   删除单个字符
  • yy  复制当前行
  • nyy  复制当前n行
  • p   粘贴
    • u  撤销操作
    • .  重复执行
    • J  连接上下行为一行
    • r   单个字符替换
  • ZZ  退出编辑，=:x
文件权限
  • chmod -R 741 dir   递归修改dir目录下的所有内容权限
  • chmod u+rx  给所属用户加上rx权限
  • chmod g-w   给用户组去掉w权限
  • chmod o=,g+w  其他用户权限为空，给当前组加w权限
  • chmod a=rw  讲所有用户权限设为rw
  • chmod 000 777 640 ...
修改文件所属用户和组
  • chown username filename 修改文件所属用户
  • chgrp groupname filename 修改文件所属组
  • chown username.groupname filename 同时修改
  • chown .groupname filename 只修改组
  • chown -R username.groupname dir 递归修改
系统常用命令

  • kill -9 pid  杀死指定pid号进程

ftp服务器配置
  • 使用root用户登陆ftp
    vim /etc/vsftpd/ftpusers
    vim /etc/vsftpd/user_list
    注释掉里面的root用户(实际中不建议这样做)
    //重启vsftpd服务
    service vsftpd restart
  • 设置 白名单 或 黑名单
    vim /etc/vsftpd/vsftpd.conf
    //修改内容
        userlist_enable=YES
        userlist_file=/etc/vsftpd/user_list
        userlist_deny=NO
    //userlist_enable=YES表示启用userlist文件来管理ftp用户登录
    //userlist_file=xxx指定userlist文件的路径(默认使用当前目录)
    //userlist_deny=NO：user_list是一个白名单，该文件中的用户允许登录
    //userlist_deny=YES(默认)：user_list是一个黑名单，该文件中的用户被拒绝登录
  • 只允许访问用户主目录
    vim /etc/vsftpd/vsftpd.conf
    //修改内容(去掉注释)
        chroot_list_enable=YES
        chroot_list_file=/etc/vsftpd/chroot_list
    //将要限制的用户添加进chroot_list

    //限定用户test不能telnet，只能ftp
