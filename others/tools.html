编程中常用的工具

  • node.js 很多软件要用到npm命令，
  • ruby 一些软件要gem安装
  • svn 局域网之间用来版本控制
  • git 任何方面都比svn好用
  • xmind 思维图，需要学习使用
  • sublime 主要代码编写工具
  • notepad++ 辅助代码工具
  • navicat 本地数据库图形界面
  • tpyora 专业markdown编辑器，不太实用
  • atom 代码编辑器，暂时还不如sublime顺手
  • hbuilder 国产前端开发工具，非常好用
  • vm 虚拟机，免费，用来学习linux
  • xshell 远程操作服务器
  • listary windows便捷启动工具
  • shadowsocks 不能说
